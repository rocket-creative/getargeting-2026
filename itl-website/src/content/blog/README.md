# Legacy Blog Content

This directory contains archived blog posts from the original genetargeting.com/ingenious-blog.

## File Format

Each blog post is a markdown file with frontmatter:

```yaml
---
title: "Post Title"
date: "YYYY-MM-DD"
category: "Technical Guide"
description: "Brief excerpt or description"
legacy_url: "https://www.genetargeting.com/ingenious-blog/post-slug"
tags: ["knockout", "cre-lox"]
---

Post content goes here in markdown format...
```

## Categories

- **Technical Guide** - In-depth technical explanations
- **Educational** - General educational content about mouse models
- **Industry Insights** - Industry news and trends
- **Company News** - Ingenious company announcements
- **Research Spotlight** - Featured research and publications
- **Selection Guide** - Help choosing between model types
- **Protocol** - Step-by-step protocols
- **General** - Other content

## File Naming

Files should be named with the URL slug:
- `how-a-knockout-mouse-is-made.md`
- `cre-lox-facts.md`
- `humanized-mice.md`

## Notes

- Content is preserved verbatim from the legacy site
- These are archived posts and may contain outdated information
- Original URLs are preserved at `/ingenious-blog/[slug]`
